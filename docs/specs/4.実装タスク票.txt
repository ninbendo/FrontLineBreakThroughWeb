# 実装タスク票 v1

## FrontLineBreakThroughWeb（MVP）

**対象**：Unity 6（6000.3.9f1） / Web（WebGL） / 縦画面 / 3D見下ろし斜め視点（3D Trigger実装）
**目的**：MVPをブラウザで遊べる状態まで作る

---

## 全体方針（先に読む）

* まずは **図形（四角・丸）でOK**
* アセット探しは後回し
* **1タスクごとに動作確認**
* エラーが出たら次へ進まない（戻って直す）
* Webビルド確認は最後にまとめて行う（途中でも可）

---

## フェーズ0：プロジェクト準備（初日）

### Task 0-1：新規プロジェクト作成

**目的**：作業用Unityプロジェクトを用意する

#### 操作（日本語UI / 英語UI）

1. Unity Hub で **新しいプロジェクト（New Project）**
2. テンプレートは以下のどちらかを選ぶ

   * **2D Core**（推奨） / **2D Core**
   * なければ **2D** / **2D**
3. プロジェクト名：`FrontLineBreakThroughWeb`
4. 保存先を確認して **作成（Create Project）**

#### 完了条件

* Unityエディタが開く
* プロジェクトが開始できる（テンプレートは2D Coreだが、実装は3D Collider / Triggerを使用する）

---

### Task 0-2：Webビルド設定の確認

**目的**：あとで詰まらないようにWeb公開設定を先に確認する

#### 操作（日本語UI / 英語UI）

1. **ファイル（File） > ビルドプロファイル（Build Profiles）**
2. 左側で **Web** を選択（Unity 6系ではWebGLが「Web」表記の場合あり）
3. **有効（Enabled）** にする
4. 上部タブ **プレイヤー設定（Player Settings）** を開く
5. `WebGL` / `Publishing Settings` 相当の項目を開く
6. **圧縮形式（Compression Format）** を **無効（Disabled）** にする

#### 完了条件

* Webが有効
* Compression Format = Disabled
